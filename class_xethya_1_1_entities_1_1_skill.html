Detailed Description

A skill is an ability performable by any entity that derives from SkilledEntity. It has the same base components of an attribute (thus, it can be altered with modifiers). Also, a skill can be affected by attributes: for instance, a "Punch" skill will be affected by a "Strength" attribute and an "Agility" attribute.
